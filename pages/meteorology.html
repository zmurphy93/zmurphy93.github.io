Numerical Weather Models

A webpage that displays model data from the GFS, NAM, and GEFS models. Created using the Flask web development framework and Python's Siphon and Metpy libraries.

Cross Sections

Displays cross-sections of gridded model data, and includes depictions of atmospheric circulations created from the solution of a second-order partial differential equation.
