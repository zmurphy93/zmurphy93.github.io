Logistic Regression

Background
Despite its name, Logistic Regression is actually a model used for classification tasks.

Building the Algorithm
A class LogisticRegression. is instantiated and contains a constructor method that will possess several hyperparameters that can be tuned by the user. These hyperparameters are:

regularization: (string)

C: The regularization coefficient.

alpha: (float) The learning rate to be used during the weight-optimization process.
